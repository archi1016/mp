顏色

由 R,G,B,A 組成

R
紅色, 0~255
G
綠色, 0~255
B
藍色, 0~255
A
透明, 0~255
當值為 128 時, G 跟 B 的值會被忽略, R 的值為 Windows 系統顏色的編號
例: 18,0,0,128

按鈕圖片

由四種狀態組成

格式
建議使用 png
狀態
  • 未使用
  • 正常
  • 駐點
  • 按下
切割
透過設定值切割重組成所需按鈕大小
效果
左圖右字

項目

FONT_SIZE
字體大小 (pt)
TEXT
文字樣板, 留白就不會顯示文字, 預設: %NAME%
TEXT_PADDING
文字左右留白 (像素)
TEXT_ALIGN
文字對齊方向

0靠左

BACKGROUND_COLOR
整個區塊的背景顏色
DRAW_OVER_COUNT
無按鈕區域描繪方式
值說明
0不描繪
1補滿按鈕
2補滿按鈕並填滿區塊

值說明

1顯示
PHOTO_WIDTH
照片寬度 (像素)
PHOTO_HEIGHT
照片高度 (像素)
BUTTON_PADDING
按鈕裡面的留白 (像素)
BUTTON_WIDTH
按鈕高度 (像素)
BUTTON_IMAGE

BUTTON_IMAGE_PADDING
