用記事本編輯 (例: D:\HTTPD\WWW\mpos\order\config.php)

MOPO_SERVER_PORT
服務端的埠號 (例: 56789)
以位址查桌號用
MOPO_FRONT_IP
收銀台的位址 (例: 127.0.0.1)
接收訂單用
MOPO_FRONT_PORT
收銀台的埠號 (例: 56788)
接收訂單用
LAN_IP_SEGMENT
區域網路網段 (例: 192.168.1)
LAN_IP_START
區域網路第一個號碼 (例: 1)
LAN_IP_END
區域網路最後一個號碼 (例: 254)
PAGE_TITLE
書籤名稱
ORDER_MODE
自助點餐模式
  • HERE - 內用先結帳, 每桌一裝置, 綁定位址
  • TAKEAWAY - 外帶或內用先結帳, 不限裝置, 不綁位址
  • TABLE - 開桌後結帳, 每桌一裝置, 綁定位址
